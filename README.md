# Mechinno Management Panel

پنل مدیریتی فارسی و راست‌به‌چپ برای مرکز نوآوری دانشکده مکانیک. PHP ساده بدون Composer، مناسب cPanel.

## ویژگی‌ها

- رابط کاربری روشن با تم آبی کم‌رنگ و فونت Vazirmatn
- **پنل خالی** — فقط ۲۴ میز از قبل ساخته می‌شود؛ بقیه داده‌ها را خودتان وارد می‌کنید
- داشبورد با خلاصه ماه، اقدامات پیشنهادی، نمودار شارژ و بدهی
- مدیریت نهادها (تیم / شرکت / دانشجو)، اعضا، میزها، کمدها، شارژ و مالی
- شناسه‌های خودکار (`T-001`, `M-001` و …)
- پروفایل کامل هر نهاد (میز، اعضا، کمد، شارژ، واریز)
- صفحه‌بندی جداول
- محاسبه خودکار شارژ از نرخ سالانه (دکمه در بخش شارژ)
- ویرایش دستی مبالغ شارژ و تراکنش‌ها
- خروجی Excel و گزارش چاپی PDF

## نیازمندی‌ها

- PHP 8.1+
- PDO (MySQL یا SQLite)
- ZipArchive و SimpleXML (فقط برای خروجی Excel)

## نصب روی cPanel

1. دیتابیس MySQL و کاربر بسازید.
2. فایل‌های پروژه را آپلود کنید.
3. `config.sample.php` را به `config.php` کپی و تنظیم کنید.
4. رمز ورود پنل را در `config.php` عوض کنید.
5. `https://your-domain.com/install.php` را باز کنید، وارد شوید و نصب را تأیید کنید.
6. پنل: `https://your-domain.com/index.php`

**نیازی به پاک کردن دستی دیتابیس نیست.** اگر داده قبلی وجود داشته باشد، نصب مجدد همه رکوردها را پاک و پنل خالی (۲۴ میز) می‌سازد.

### بعد از نصب — فایل‌های قابل حذف

| حذف کنید | دلیل |
|----------|------|
| `install.php` | جلوگیری از نصب مجدد تصادفی |

| نگه دارید | دلیل |
|-----------|------|
| `config.php` | اتصال دیتابیس و ورود |
| بقیه فایل‌های پنل | اجرای عادی |

## شروع کار در پنل خالی

1. **نهادها** — تیم، شرکت یا دانشجو ثبت کنید
2. **اعضا** — هر عضو را به نهاد و میز(ها) وصل کنید
3. **کمدها** — شماره کمدها را خودتان اضافه کنید
4. **شارژ** — نرخ سالانه تعریف کنید، سپس محاسبه خودکار یا ثبت دستی
5. **مالی** — درآمد، هزینه و دریافت شارژ از نهادها (مرکز طلبکار، نهاد بدهکار)

برای پاک کردن همه داده و شروع دوباره: **بازنشانی پنل** در منو یا `install.php`.

## مسیرهای اصلی

| مسیر | کاربرد |
|------|--------|
| `index.php` | پنل اصلی |
| `install.php` | نصب / بازنشانی پنل خالی |
| `api.php?resource=summary` | API داشبورد |
| `api.php?resource=team-profile&id=1` | پروفایل نهاد |
| `POST api.php?resource=recalculate-charges` | محاسبه خودکار شارژ |
| `export.php?report=all` | خروجی Excel |
| `report.php` | گزارش مالی قابل چاپ |

## API صفحه‌بندی

```text
GET api.php?resource=members&page=2&per_page=25
```

پاسخ:

```json
{
  "rows": [...],
  "total": 0,
  "page": 2,
  "per_page": 25,
  "pages": 1
}
```

## شارژ و نرخ

- نرخ‌ها در بخش **شارژ → نرخ‌های سالانه** تعریف می‌شوند.
- مبالغ شارژ را می‌توان دستی ویرایش کرد.
- دکمه **محاسبه خودکار از نرخ** شارژ ماهانه را از تعداد میز × نرخ سالانه محاسبه می‌کند.

## امنیت

`.htaccess` دسترسی به `config.php`، `src/` و `data/` را محدود می‌کند. رمز قوی برای پنل و دیتابیس انتخاب کنید.
